## 🔧 Решение проблем

### Ошибка: "Port 5000 already in use"
```bash
# Используйте другой порт (отредактируйте server-standalone.js, строка 95)
# const PORT = 5001;  # Измените на 5001 или другой номер
```

### Ошибка: "Cannot find module"
```bash
# Убедитесь, что все файлы есть:
ls -la "e:\Aiprojects\CRM 2026"

# Все файлы должны быть в этой папке
```

### Сайт не анализируется
- Проверьте URL (должен быть https:// или http://)
- Убедитесь, что сайт доступен в браузере
- Попробуйте другой сайт

## 🎓 Для разработчиков

### Структура проекта
```javascript
// server-standalone.js
- parseHtml()      // Парсинг HTML
- fetchPage()      // HTTP запрос
- analyzeSite()    // Комплексный анализ
- /api/analyze     // API endpoint
```

### Как добавить новую метрику
1. Отредактируйте функцию `parsehHtml()` в `server-standalone.js`
2. Добавьте новое поле в объект `result`
3. Обновите `index.html` для вывода
4. Добавьте обработку в `client.js`
