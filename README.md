# SEO Audit Pro — Бесплатный SEO-аудит и анализ конкурентов

Полнофункциональное веб-приложение для анализа SEO вашего сайта и сравнения с конкурентами. Все данные собираются через парсинг HTML и анализ контента — **никаких платных API не требуется**.

## 🚀 Быстрый старт

### Вариант 1: Облегченная версия (БЕЗ npm — РЕКОМЕНДУЕТСЯ)

```bash
# Запустите сервер без установки зависимостей
node server-standalone.js
```

Готово! Приложение доступно по адресу **http://localhost:5000**

---

### Вариант 2: Полная версия (с Express и Cheerio)

Требует установку npm зависимостей:

```bash

npm install

# 2. Запустите сервер

# 3. Откройте в браузере
http://localhost:5000
```

**Внимание:** Если у вас проблемы с npm, смотрите [SETUP_HELP.md](SETUP_HELP.md)

## 📋 Функционал

### ✅ Полный SEO-анализ вашего сайта:
- **Title, Meta Description, Meta Keywords** — проверка обязательных тегов
- **Заголовки (H1-H6)** — анализ структуры документа
- **Индексация** — проверка noindex/nofollow
- **Robots.txt и Sitemap.xml** — наличие файлов (требуется проверка вручную)
- **Canonical URLs** — обнаружение дублей контента
- **Alt-теги у изображений** — процент заполненности
- **Внутренние и внешние ссылки** — анализ ссылочной структуры
- **Friendly URLs** — проверка человекопонятных URL

### ⚙️ Технический анализ:
- **HTTP статус** — проверка доступности (200, 404, 5xx и т.д.)
- **SSL/HTTPS** — наличие защиты
- **Время загрузки** — скорость ответа сервера
- **Размер страницы** — объем HTML
- **Gzip сжатие** — оптимизация передачи
- **Адаптивный дизайн** — проверка viewport meta-тега
- **Media queries** — поддержка мобильных устройств
- **Количество скриптов и стилей** — анализ ресурсов

### 👥 UX и дизайн:
- **Контактные данные** — телефон, email, адрес
- **Кнопки действия (CTA)** — поиск слов "купить", "заказать", "оставить заявку"
- **Форма обратной связи** — наличие и анализ
- **Навигация** — меню, поиск, хлебные крошки
- **Контрастность** — базовая проверка читаемости

### 💰 Конверсионные элементы:
- **Email форма** — для сбора подписок
- **Online чат** — Livechat, Tidio, Jivosite и т.д.
- **Отзывы и кейсы** — социальное доказательство

### 📈 Ключевые метрики:
- **Количество слов** — объем контента
- **Топ-5 ключевых фраз** — часто повторяющиеся триграммы
- **Плотность ключевых слов** — основной показатель
- **Анализ слов** — топ-5 наиболее частотных слов

## 🏆 Анализ конкурентов

1. **После анализа вашего сайта** приложение предложит добавить конкурентов
2. **Добавьте до 3 конкурентов** — введите их URL
3. **Получите сравнительную таблицу** с ключевыми метриками
4. **Автоматические рекомендации** — что взять у конкурентов, что улучшить

## 📊 Экспорт отчётов

- **JSON** — полный отчёт со всеми деталями
- **CSV** — таблица для Excel/Google Sheets

## 🛠️ Технический стек

- **Backend:** Node.js + Express
- **Парсинг:** Cheerio (быстрый парсер HTML)
- **HTTP запросы:** Axios
- **CORS:** Встроенная поддержка
- **Frontend:** Чистый JavaScript (без фреймворков)
- **Стили:** Bootstrap 5 CDN + собственный CSS

## ⚠️ Ограничения и заглушки

### Позиции в поиске (Google/Яндекс)
❌ **Невозможно получить бесплатно** — требуется платное API  
💡 Рекомендуем проверить вручную через:
- Google Search Console (бесплатно, для своего сайта)
- Serpstat (триал)
- Semrush (триал)

### Скорость загрузки
⚠️ **Зависит от прокси** — измеряется время ответа сервера вашего хостинга  
📌 Реальная скорость для пользователя может быть выше/ниже в зависимости от:
- Расстояния до сервера
- Качества интернета
- Оптимизации браузером

### Анализ дизайна
📋 **Только по наличию элементов** — без ИИ-оценки качества

### Поиск конкурентов
🔍 **Ручное добавление** — приложение предлагает топ ключевые слова для поиска

## 🐛 Обработка ошибок

### Ошибка: "Сайт не доступен"
- Проверьте правильность URL (нужно https:// или http://)
- Убедитесь, что сайт доступен в браузере
- Попробуйте другой сайт

### Ошибка: "Таймаут"
- Сервер слишком долго отвечает
- Попробуйте позже или проверьте интернет

### Ошибка: "Ошибка CORS"
- Приложение использует свой прокси — такой ошибки не должно быть
- Если возникла, перезапустите сервер
