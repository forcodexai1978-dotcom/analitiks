## ⚠️ ВОЗМОЖНЫЕ ОШИБКИ И РЕШЕНИЕ

### Ошибка: "Сайт не доступен"
```
Причина: Сайт не отвечает или неверный URL
Решение: Проверьте URL и интернет-соединение
```

### Ошибка: "Таймаут"
```
Причина: Сайт слишком долго загружается
Решение: Попробуйте другой сайт или подождите
```

### Ошибка: "Невозможно подключиться к http://localhost:5000"
```
Причина: Сервер не запущен
Решение: 
1. Откройте новый терминал
2. Выполните: cd "e:\Aiprojects\CRM 2026"
3. Выполните: node server-standalone.js
4. Обновите браузер (Ctrl+R)
```

### Сайт не меняется при обновлении
```
Причина: Кэш браузера
Решение: Нажмите Ctrl+Shift+Delete (очистить кэш)
```
